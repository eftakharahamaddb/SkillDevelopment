# RxJava

### Sources :
**Observable\<T\>**    
  1) 0 or n items
  2) Complete or error<br>
  3) Does not has backpressure
  
**Flowable\<T\>**
  1) 0 or n items
  2) Complete or error
  3) Has backpressure
  
  >Backpressure Means:It allows to controle how fast source can be emits data.
